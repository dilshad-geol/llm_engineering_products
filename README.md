# LLM Engineering — Products & Experiments

This repository collects **products** and **experiments** built while working in **LLM engineering**: designing, shipping, and evaluating systems that use large language models in real workflows.

## What you’ll find here

Topics and project types may include:

- **Chatbots & conversational agents** — multi-turn dialogue, tool use, guardrails, and UX around LLM-backed assistants  
- **RAG (retrieval-augmented generation)** — chunking, embeddings, vector stores, re-ranking, evaluation, and production retrieval patterns  
- **Prompting, evaluation & observability** — test sets, quality metrics, tracing, and iteration loops  
- **Other LLM-adjacent builds** — agents, structured outputs, fine-tuning or alignment experiments, and integrations with APIs and data pipelines  

Individual folders or subprojects will describe their own setup, dependencies, and how to run them.

## Structure

As projects are added, each one typically lives in its own directory with a short description at the top of its readme or module docstring. This root readme stays a high-level map of the repo.
